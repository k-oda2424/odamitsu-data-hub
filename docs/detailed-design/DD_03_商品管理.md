### 5.3 UPSERTロジック詳細（MSalesGoodsService#save）

```java
public MSalesGoods save(ISalesGoods iSalesGoods) throws Exception {
    // Step1: PK検索
    MSalesGoods salesGoods = this.getByPK(
        iSalesGoods.getShopNo(), iSalesGoods.getGoodsNo());

    // Step2: 分岐
    if (salesGoods == null) {
        // INSERT: 新規MSalesGoods生成、modifyDateTime/modifyUserNoをnull化
        return this.insert(iSalesGoods);
    }
    // UPDATE: BeanUtils.copyPropertiesで全フィールド上書き
    return this.update(iSalesGoods);
}
```
